# 成长型思维

## 思想来源
源自卡罗尔·德韦克(Carol Dweck)的研究，在《心态》(Mindset, 2006)中系统阐述。

## 一句话定义
你的能力不是固定的——它是可以通过努力和学习不断增长的。把"失败"当成"还没学会"的信号，而不是"我不行"的证据。

## 核心逻辑

### 两种心态

| | 固定型思维 | 成长型思维 |
|------|-----------|-----------|
| 核心信念 | 能力是天生的，后天很难改变 | 能力是可以通过努力和学习增长的 |
| 对失败的反应 | "我不行" | "我还没学会" |
| 对挑战的态度 | 避免挑战（怕暴露自己不行） | 迎接挑战（挑战是成长的机会） |
| 对反馈的反应 | 防御、回避 | 接受、改进 |
| 对他人成功的反应 | 威胁、嫉妒 | 学习、借鉴 |

### 关键洞察
- 固定型思维不是说人不想成长——而是说人**认为**自己无法成长
- 改变心态不是说"努力就能做任何事"——而是说"通过正确的努力，我可以在很多方面做得更好"
- 心态不是天生的——它是可以通过引导改变的

### 成长型思维的边界
- 不是"任何人都能成为爱因斯坦"——成长型思维说的是在你现有的基础上进步
- 不是"不需要天赋"——天赋和努力的结合才是最优解
- 不是"努力就够"——需要"有效的努力"——在错误的方向上努力是无效的

## 操作步骤

### 步骤1：识别你当前的思维模式
问自己：
- 当我遇到困难时，我第一反应是"我不行"还是"我还没学会"？
- 当我看到别人比我强时，我是感到威胁还是好奇？

### 步骤2：改变你的"语言"
固定型思维和成长型思维的核心区别在语言：
- 把"我做不了这个"换成"我还没学会这个"
- 把"这个人比我强，我不如TA"换成"这个人是怎么做到的？我能学到什么？"
- 把"我失败了"换成"我学到了什么？"

### 步骤3：用"还没"(Yet)的力量
在否定句后面加一个"yet"：
- "我不懂这个" → "我还不懂这个**yet**"
- "我做不好" → "我还做不好**yet**"
- 这个小小的语言改变会产生巨大的心理效果

### 步骤4：表扬过程，不表扬天赋
如果你在教育孩子或管理团队：
- 不要说"你真聪明"（固定型思维——暗示能力是天生的）
- 要说"你这次真的很努力"（成长型思维——暗示努力带来进步）

### 步骤5：寻找有效的学习策略
成长型思维 + 盲目努力 = 无效
成长型思维 + 有效策略 = 进步
找到高效的学习方法比单纯"更努力"重要得多。

## 参考案例
德韦克最经典的实验：给一群中学生做简单的测试。一半学生被表扬"你很聪明"（固定型），另一半被表扬"你很努力"（成长型）。之后给他们一个更难的测试——被表扬"聪明"的学生大幅退缩，被表扬"努力"的学生反而更投入。更关键的是后续测试：回到简单难度后，"聪明组"的分数比之前**降低了**，而"努力组"的分数比之前**提高了**——一次错误的表扬方式，不仅影响当下的表现，还影响了未来的表现。

## 适用条件
- 特别有效：教育（特别是儿童和青少年）、个人学习、职业转型期、团队文化建设
- 需谨慎：在需要快速证明能力的场景中（面试、比赛），成长型思维不能代替实际准备
- 可能误导：把"成长型思维"变成"你不够努力"——有时候问题不在努力，在方法和方向

## 局限性
- 心态改变需要时间，不是一次就能改过来的
- 环境对心态的影响很大——在一个只看结果的团队中，很难维持成长型思维
- 有些能力的上限确实存在——成长型思维说的是"进步"，不是说"无限进步"

## 相关方法论
- 习得性乐观（成长型思维是习得性乐观的基础）
- 反脆弱（成长型思维本身就是一种反脆弱能力）
- PERMA模型（成长型思维影响A-成就和E-投入维度）
- 认知偏差（固定型思维本身就是一种认知偏差）
