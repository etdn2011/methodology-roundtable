## [0.1.0] - 2026-07-18

### 初始版本发布

#### 新增
- **方法论圆桌系统** (`SKILL.md`) — 完整的圆桌分析流程引擎（6步流程：问题诊断→强制搜索→方法论选择→角色配置→逐位发言→交叉讨论→汇总输出）
- **角色定义库** (`roles/SKILL.md`) — 5个分析角色：A看山（战略）、B修路（执行）、C织网（资源）、D照镜（批判）、E种树（人文）
- **财富类方法论×8**:
  - `flywheel-effect` — 飞轮效应（正反馈循环分析）
  - `jobs-to-be-done` — JTBD（用户任务洞察）
  - `concentration-principle` — 压强原则（资源集中突破）
  - `grey-zone-management` — 灰度管理（动态平衡决策）
  - `growth-hacking` — 增长黑客（实验驱动增长）
  - `long-termism` — 长期主义（长期投入评估）
  - `positioning-theory` — 定位理论（心智占位分析）
  - `rpv-framework` — RPV框架（组织能力诊断）

#### 设计原则
- 零预置案例 — 所有论据来自实时 WebSearch
- 零人物绑定 — 方法论标注思想来源，不扮演任何人
- 操作步骤化 — 每个方法论像"使用说明书"
- 说给普通人听 — 避免术语堆砌
- 用户确认前置 — 问题不充分时不猜测，先确认

#### 技术栈
- 纯 Markdown + YAML frontmatter（TRAE Skill 格式）
- 依赖 TRAE 的 WebSearch 工具获取实时资讯
- 独立 skill 架构，每个方法论可单独调用
